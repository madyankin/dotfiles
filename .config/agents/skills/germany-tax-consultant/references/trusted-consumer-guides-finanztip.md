# Trusted Consumer Guides: Finanztip

Use this reference only after checking official sources or when the question is about practical consumer guidance rather than raw law.

## Sources

- `https://www.finanztip.de/steuererklaerung/`
- `https://www.finanztip.de/elster/`
- `https://www.finanztip.de/steuererklaerung/steuererklaerung-pflicht/`
- `https://www.finanztip.de/steuererklaerung/steuererklaerung-frist/`

## What To Use These For

- plain-language filing overviews
- practical ELSTER expectations for normal users
- summaries of filing obligation patterns
- deadline reminders and practical framing

## What Not To Use These For

- overruling official agency guidance
- proving a contested legal point
- giving definitive answers in edge cases

## Source Priority Rule

If a Finanztip explanation differs from BZSt, ELSTER, BMF, or statute text, prefer the official source and explain the simpler practical takeaway in your own words.

## Style Guidance

- Borrow the clarity, not the authority.
- Do not copy consumer-guide wording as if it were law.
- Use these guides to make answers easier to understand after the official answer is established.
